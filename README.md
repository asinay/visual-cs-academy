# Visual CS Academy

![Visual CS Academy logo](assets/logo.png)

Visual CS Academy is a visual-first, self-contained Computer Science study platform built for students preparing for exams.

The current milestone is a polished Data Structures study guide delivered as a single `index.html` app with no backend and fast GitHub Pages hosting.

Question authoring can also happen in a separate repo file, [EXAM_BANK.md](/C:/AI_Apps/visual-cs-academy/EXAM_BANK.md), while the shipped app remains self-contained in `index.html`.

## Local Development

Open `index.html` directly in a browser. No build step or backend is required.

## Exam Authoring

Use [EXAM_BANK.md](/C:/AI_Apps/visual-cs-academy/EXAM_BANK.md) to refine question difficulty, draft stronger scenario questions, and plan future quizzes without editing the runtime app immediately.

Current workflow:

- Keep the live site self-contained in `index.html`
- Draft or revise exam questions in `EXAM_BANK.md`
- Promote approved questions into `index.html` when ready
