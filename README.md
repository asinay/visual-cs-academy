# 🎓 Visual CS Academy

> Learn Computer Science visually.

Visual CS Academy is an interactive, visual-first learning platform designed to help students master Computer Science concepts through concise explanations, memorable analogies, interactive quizzes, and curated visual resources.

The project began as a study companion for my daughter as she prepared for her college Data Structures exam. Rather than replacing great educational resources already available online, it brings them together into a single, easy-to-follow learning experience.

## ✨ Features

* 🎨 Visual-first learning experience
* 📚 Concise theory focused on understanding, not memorization
* 🧠 Memory techniques and analogies
* ✅ Interactive quizzes with instant feedback
* ✔️ True/False and multiple-choice questions
* 🎯 Mixed "exam mode" quizzes
* 📈 Progress tracking
* 🌐 Curated links to high-quality external visualizations (such as VisuAlgo and USFCA)
* 📱 Responsive design for desktop, tablet, and mobile
* 🚀 Runs entirely in a single HTML file — no installation or backend required

## 🎯 Project Goals

The goal is not simply to explain Computer Science concepts.

The goal is to help students:

* Understand **when** to use each data structure
* Recognize common exam questions
* Build lasting mental models
* Develop intuition rather than memorizing definitions

## 🚀 Running the App

Simply open `index.html` in any modern web browser.

No installation, dependencies, or build process required.
