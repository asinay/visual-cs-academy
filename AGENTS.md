# Visual CS Academy

## Repository

https://github.com/asinay/visual-cs-academy

Live site:
https://asinay.github.io/visual-cs-academy/

## Mission

Build the best visual-first Computer Science study platform.

The first course is Data Structures.

Primary audience:
College students preparing for exams.

## Current milestone

Deliver a polished Data Structures study guide.

## Design principles

- Visual learners first
- Mobile friendly
- Self-contained HTML
- No backend
- Fast loading
- Professional appearance
- Clear over clever

## Preferred learning flow

Overview
↓
Theory Snapshot
↓
Memory Tricks
↓
Visual References
↓
Cheat Sheet
↓
Exam Traps
↓
Quiz
↓
Final Exam

## Quiz rules

- Mix multiple choice and True/False
- Explain every answer
- Include scenario-based questions
- Randomize question order where possible

## Code rules

Keep everything inside `index.html`.

Avoid frameworks unless explicitly requested.

Never reduce existing functionality while making improvements.
